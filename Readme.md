# My Home

## Features
* Firebase
* Angular Fire
* Typescript
* Material Design


### Update Bower components

`bower install -f --save-dev`

### Install Typedefintions

`tsd install <package> --save `

### Update Typedefintions

`tsd update --save --overwrite`


### Install Live Server
`npm install live-server -g`


### Run Live Server

Goto to the app folder (i.e. public)

`live-server --quiet`
